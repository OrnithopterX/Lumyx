# Lumyx
An open-source LiDAR based 3D scanning platform focused on accessibility, flexibility, and ease of use.

Lumyx (loom - ex) captures the last year of my efforts to make a truly accessible 3D scanning system. All the software is open-source, and the machinical parts are largly 3D printable. This is just the beginning!
