# Lumyx
An open-source LiDAR based 3D scanning platform focused on accessibility, flexibility, and ease of use.

Lumyx (loom - ex) captures the last year of my efforts to make a truly accessible 3D scanning system. All the software is open-source, and the machinical parts are mostly 3D printable. This is just the beginning! 

<img width="1242" height="728" alt="LUMYX" src="https://github.com/user-attachments/assets/f44534ed-e0ff-40ed-b784-4837672e35b6" />

The generated point-clouds can be opened in Blender for viewing. This example uses multiple scans stitched together to produce this result.
